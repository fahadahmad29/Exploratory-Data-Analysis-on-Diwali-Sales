# Exploratory Data Analysis on Diwali Sales

This project analyzes Diwali sales data using Python to identify customer purchasing patterns, top-selling product categories, and regional sales trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains 11,251 records and 15 features related to customer demographics and purchase behavior.

## Key Insights
- Identified top-performing product categories
- Analyzed purchasing behavior by gender and age group
- Examined regional sales distribution

## Project Files
- EDAproject.ipynb – Full analysis notebook
- Diwali_Sales_Data.csv – Dataset
